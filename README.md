# きょうのよていボード v3

## 変更点

1. 左端の「あさ／ひる／よる」も絵カードから変更可能
2. 予定カードをドラッグして移動・入れ替え可能
3. カード一覧は `cards/cards.json` で管理

## 使い方

- 左端の場面カードをクリックすると、その行のラベルカードを変更できます。
- 空き枠または予定カードをクリックすると、絵カードを選べます。
- 予定カードはドラッグして別の枠に移動できます。
- 予定カードを別のカードの上に置くと、カードが入れ替わります。
- 「できたチェック」ONのときは、ドラッグではなく完了チェックになります。

## カード追加方法

1. `cards` フォルダに画像を追加
2. `cards/cards.json` に追記

例：

```json
{
  "id": "hair_cut",
  "label": "かみをきる",
  "image": "hair_cut.png"
}
```

## ファイル構成

```text
index.html
cards/
  cards.json
  morning.svg

  night.svg
  ...
README.md
```
